# Road Damage Segmentation using Image Processing

## Overview
This project focuses on detecting and segmenting different types of road damages using image processing techniques and OpenCV. The system processes road video frames and identifies damages such as potholes, cracks, and alligator cracks through preprocessing, segmentation, contour detection, and classification methods.

The project was developed as part of an academic image processing assignment.

---

## Objectives
- Detect road damages from road surface videos
- Segment damaged regions using image processing techniques
- Classify multiple road degradation types
- Evaluate segmentation performance using standard metrics
- Generate visual outputs for analysis and reporting
