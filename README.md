# Road Surface Crack Detection using Image Enhancement

## Project Description
This project focuses on improving the visibility of road surface cracks using image processing techniques. 
A video was recorded using a mobile phone while moving on a road during daytime. Frames were extracted 
from the video and image enhancement methods were applied to improve crack detection.

## Methods Used
- Frame Extraction from video
- Noise Reduction using Median Filter
- Illumination Enhancement using CLAHE
- Blur Analysis using Laplacian Variance
- Histogram Analysis

## Tools and Libraries
- Python
- OpenCV
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset
Mobile phone video recorded during daytime road travel.

## Results
Image enhancement improved contrast and reduced noise, making road cracks more visible for segmentation.
